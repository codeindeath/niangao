# 年糕管理后台 需求框架 v0.2

## 一、总览仪表盘

### 1.1 核心指标卡片
- 总用户数（实时）
- 总经验数（实时）
- 今日新增用户
- 今日新增经验
- 今日活跃用户（有浏览/点赞/收藏任一行为）
- 今日 AI 对话次数

### 1.2 审核管道概览
- 待审核数量
- 今日审核通过数
- 今日审核拒绝数
- 积压数量（超过 1 小时未审）

### 1.3 AI 服务状态
- DeepSeek API 连通性（绿/黄/红灯）
- 近 1 小时 API 调用量
- 近 1 小时 API 失败率
- 平均响应延迟

### 1.4 趋势迷你图
- 近 7 天新增用户曲线
- 近 7 天新增经验曲线
- 近 7 天 活跃用户曲线

### 1.5 快捷操作入口
- 一键跳转审核队列
- 一键跳转用户列表

---

## 二、用户管理

### 2.1 用户列表
- 分页展示（头像、昵称、登录方式、称号、注册时间、状态）
- 按昵称搜索
- 按用户 ID 搜索
- 按登录方式筛选（Apple / Dev）
- 按注册时间范围筛选
- 按状态筛选（正常 / 已禁用）
- 按有无称号筛选
- 排序（注册时间 / 经验数 / 活跃度）
- 列表导出 CSV

### 2.2 用户详情
- 基本信息：ID、昵称、头像、称号、简介、注册时间、最近登录
- 登录信息：登录方式、Apple 账号邮箱（如有）
- 个人统计：发布数 / 获点赞数 / 被收藏数 / 浏览数 / 我点赞数 / 我收藏数 / 对话次数 / 消息条数
- 领域偏好分布（发布×2 + 收藏×1 权重图）

### 2.3 用户经验列表
- 该用户发布的所有经验（含私密、未通过、已删除）
- 每条显示：内容摘要、领域、审核状态、发布时间
- 点击跳转该条经验详情

### 2.4 用户状态管理
- 禁用用户（禁止登录 + 禁止发布，已发布内容保留）
- 启用用户（恢复）
- 批量禁用/启用
- 禁用理由记录（内部备注）

---

## 三、内容审核

### 3.1 审核队列
- 待审核经验列表（按提交时间升序，最旧的优先）
- 每条展示：内容摘要、领域、提交时间、AI 判定结论（通过/拒绝）、AI 置信度分数
- 高亮标记：被硬策略拒绝的（红色）、AI 判定拒绝的（橙色）
- 分页、每页数量可调

### 3.2 审核筛选
- 按一级领域筛选
- 按来源类型筛选（平台 / UGC）
- 按 AI 判定结果筛选（通过 / 拒绝 / 未判定）
- 按提交时间范围筛选

### 3.3 经验详情（审核视角）
- 完整经验内容
- AI 六维打分明细（价值/可执行/普适/原创/清晰/结构，含各维度分数和总评）
- 硬策略检查结果（哪个规则触发了拒绝）
- AI 解读内容
- 经验元信息：创建者名、来源标签、是否私密

### 3.4 单条审核操作
- 通过（覆盖 AI 判定）
- 拒绝（填写拒绝理由，覆盖 AI 判定）
- 退回重审（强制重新调用 AI 审核）
- 审核备注（内部备注，用户不可见）

### 3.5 批量审核操作
- 勾选多条
- 批量通过
- 批量拒绝（统一理由）
- 全选当前页

### 3.6 AI 打分复盘
- 查看历史 AI 打分记录
- 标记 AI 误判（误通过/误拒绝）
- 查看标记统计（用于调优 prompt）

### 3.7 审核日志
- 每条经验的审核时间线：提交 → AI 审核 → 人工操作
- 记录操作人、操作类型、时间、理由

---

## 四、内容管理

### 4.1 全量经验列表
- 分页展示所有经验（含各种状态）
- 展示：内容摘要、作者、领域、审核状态、发布状态、互动数（点赞/收藏/浏览）、发布时间
- 按内容关键词搜索
- 按创建者名/昵称搜索
- 按领域筛选
- 按来源类型筛选（平台 / UGC）
- 按审核状态筛选（待审/通过/拒绝/私密）
- 按发布时间范围筛选
- 按互动量排序（最多点赞/最多收藏）
- 导出 CSV

### 4.2 经验详情
- 完整内容 + AI 解读
- 互动数据（点赞数、收藏数、浏览数）
- 审核信息（AI 结论、人工复核记录）
- 作者信息（点击跳转用户详情）

### 4.3 经验编辑
- 修改经验内容
- 修改领域（一级 + 二级）
- 修改来源标签（平台/用户）
- 修改创建者名称
- 修改打分理由（≤15 字）

### 4.4 经验状态管理
- 软删除（已入池经验：标记删除，前端不展示但数据保留）
- 恢复软删除
- 硬删除（未入池经验：物理删除）
- 强制下架（从推荐池移除但保留）
- 修改审核状态（手动改为通过/拒绝/私密）

### 4.5 批量操作
- 批量修改领域
- 批量下架
- 批量软删除

### 4.6 互动数据管理
- 查看某条经验的点赞用户列表
- 查看某条经验的收藏用户列表
- 查看某条经验的浏览用户列表
- （暂无评论管理，预留）

---

## 五、平台内容管理（官方经验 / 种子数据）

### 5.1 平台经验列表
- 所有 source_type = platform 的经验
- 展示：内容摘要、创作者名、领域、打分/理由、发布时间
- 筛选：领域、有无 AI 解读、有无打分
- 搜索：创作者名、内容关键词

### 5.2 新建平台经验
- 填写经验内容（10-100 字）
- 选择领域（一级 + 二级）
- 填写创作者名称
- 填写来源标签（如"张小龙·微信产品观"）
- 手动填写打分理由（≤15 字）
- 保存草稿 / 直接发布

### 5.3 编辑平台经验
- 同普通经验编辑 + 可修改创作者名和来源标签

### 5.4 批量导入
- 上传 CSV/JSON 文件
- 预览导入数据（验证领域、字数限制）
- 确认导入
- 导入结果报告（成功/失败条数、失败原因）

### 5.5 批量 AI 处理
- 选中未评分的平台经验 → 一键批量调用 AI 打分+解读
- 处理进度条展示
- 处理结果报告
- 失败项重试

### 5.6 单条重打分
- 对已评分的平台经验 → 重新调用 AI 打分（覆盖原分数）

---

## 六、领域与标签管理

### 6.1 一级领域列表
- 展示所有一级领域（career / cognition / life / relationship / wealth）
- 每项显示：名称、显示名（中文）、图标/emoji、下属子领域数、经验总数
- 排序：默认排序，支持拖拽调整

### 6.2 一级领域编辑
- 新增一级领域
- 编辑显示名
- 设置图标/emoji
- 禁用（禁用后该领域不可选，已有经验不受影响）

### 6.3 二级领域列表
- 分组展示（按父领域）
- 每项显示：名称、显示名、所属父领域、对应经验数
- 排序：组内拖拽调整

### 6.4 二级领域编辑
- 新增子领域（绑定父领域）
- 编辑显示名
- 修改所属父领域（迁移已有经验）
- 禁用/启用

## 七、AI 服务管理

### 7.1 服务状态

### 7.3 批量任务管理
- 批量打分任务列表
- 查看任务进度（已完成/总数）
- 取消进行中任务
- 重试任务中的失败项
- 查看任务详细日志

### 7.4 费用估算
- 按接口维度的 Token 消耗统计
- 费用估算（需配置单价）
- 日/月费用趋势

### 7.5 Prompt 查看
- 查看当前审核 prompt 模板
- 查看解读生成 prompt 模板
- 查看对话系统 prompt 模板
- （编辑和热更新为后续阶段，此处只读）

## 九、系统配置

### 9.1 审核设置
- 审核模式：全自动（AI 判定即最终）/ 人工复核（AI 判定后需人工确认）
- 硬策略开关：逐条开关（如：禁用"重复内容"检测）

### 9.2 内容限制
- 经验字数：下限（默认 10）/ 上限（默认 100）
- 解读字数上限（默认 300）
- 称号字数上限（默认 15）

### 9.3 频率限制
- 单人每日发布上限
- 单人每日 AI 对话轮次上限
- 全局 AI API 调用频率限制

### 9.4 敏感词管理
- 敏感词列表（表格展示）
- 新增敏感词
- 删除敏感词
- 批量导入敏感词
- 开关：启用/禁用敏感词检查

### 9.5 功能开关
- 注册开关（关闭后新用户无法登录）
- AI 解读生成开关
- 搜索功能开关

### 9.6 登录方式设置
- 查看当前启用的登录方式（Apple）
- Dev Login 开关（生产环境强制关闭）

## 十、操作日志

### 10.1 管理员操作日志
- 记录所有后台操作：谁、什么时间、什么操作、操作对象、操作结果
- 操作类型覆盖：审核/编辑/删除/禁用/配置变更/批量操作
- 筛选：操作人、操作类型、时间范围、操作对象

### 10.2 敏感操作高亮
- 删除经验、禁用用户、修改系统配置 → 日志标红

### 10.3 日志保留
- 保留周期（如 90 天）
- 日志导出

### 10.4 管理员管理
- 管理员账号列表（预留，单人管理阶段一个账号即可）
- 新增/删除管理员
- 修改密码
