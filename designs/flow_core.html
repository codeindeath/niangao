年糕 App · 核心业务流程

三个核心模块的端到端数据处理链路

📝

经验发布 → AI 解读生成

用户创建内容，AI 自动生成深层解读

用户输入 100 字经验
→
选择领域分类
→
POST /experiences
→
Go 后端写入 PostgreSQL
→
异步调用 AI 服务
→
DeepSeek 生成 500 字解读
→
存回数据库 + 返回前端
❤️

社区互动（点赞 / 收藏）

Toggle 机制 + 数据库触发器自动维护计数

用户点击点赞/收藏
→
POST /experiences/:id/like
→
Go 后端 Toggle 记录
→
数据库触发器自动更新计数
→
前端乐观更新 UI
💬

AI 对话辅导

基于用户经验库的个性化人本主义对话

用户发送消息
→
AI 服务 Embedding 检索
→
匹配用户最相关的 5 条经验
→
构建人本主义 System Prompt
→
DeepSeek 流式生成回复
→
SSE 推送前端实时展示
💡 所有用户数据存储在火山引擎 RDS PostgreSQL（国内节点），AI 服务使用 DeepSeek API（国内可直接访问），确保全国用户低延迟体验。
