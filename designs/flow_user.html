年糕 App · 用户交互流程

从启动到核心功能使用的完整路径

1

启动 App

检查本地 Token，判断登录状态

已登录 → 进入首页
未登录 → 显示登录页
2

微信一键登录

用户点击"微信一键登录" → 调起微信 App 授权 → 获取 code → 后端换取 openid → 签发 JWT → 进入首页

3

首页信息流

浏览经验卡片流（下拉刷新 + 无限滚动），每张卡片展示：作者头像昵称、领域标签、100 字经验内容、点赞/收藏按钮

4

发布经验

点击底部中央 ⊕ 按钮 → 输入 100 字经验内容 → 选择领域分类 → 发布 → AI 自动生成 500 字解读

5

AI 对话辅导

进入「对话」Tab → 向 AI 辅导员发送消息 → AI 检索用户的经验库（已创建+已收藏）→ 基于人本主义方法生成个性化回复

6

个人主页

查看个人信息、已发布经验列表、已收藏经验列表、已实践标记
