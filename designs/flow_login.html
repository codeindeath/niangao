年糕 App · 微信 OAuth 登录流程

基于微信开放平台移动应用 SDK 的认证流程

1

用户点击"微信一键登录"

App 调用微信 SDK 的 SendAuthReq，请求用户授权

2

微信 App 展示授权页

用户在微信内确认授权，微信 SDK 回调返回 authorization code

3

App 将 code 发送到后端

POST /api/v1/auth/wechat/login { code }

📱 移动端

保存 JWT Token 到本地存储
保存用户信息（昵称、头像）
后续请求自动附带 Authorization 头
Token 过期 → 重新走登录流程

🖥 后端服务

调用微信 API 以 code 换 openid
查找或创建用户记录（users 表）
签发 JWT Token（7 天有效）
生成 Refresh Token
4

登录成功，进入 App 主界面

用户信息存入 AsyncStorage，所有 API 请求携带 Bearer Token
